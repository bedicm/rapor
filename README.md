# Sipariş Takip Otomasyonu - Proje Blogu

Bu depo, **Sistem Analizi ve Tasarımı** dersi kapsamında geliştirilen **Döner Salonu Sipariş Takip Otomasyonu** projesinin haftalık ilerleme raporlarını içermektedir.

## Proje Hakkında
Bu proje, bir döner salonu işletmesinde sipariş alma süreçlerini dijitalleştirmeyi ve garsonlar ile kasa arasındaki iletişimi gerçek zamanlı hale getirmeyi amaçlayan bir masaüstü otomasyon sistemidir.

### Temel Özellikler
- **Garson Mobil Arayüzü:** Sipariş girişi ve masa yönetimi.
- **Kasa Takip Paneli (Dashboard):** Sipariş onayı, ödeme ve mutfak hazırlık süreci takibi.
- **Menü Yönetimi:** Esnek ürün ve fiyat yönetimi.
- **Raporlama:** Günlük ve haftalık sipariş analizleri.

### Kullanılan Teknolojiler
- **Kullanıcı Arayüzü:** Java Swing
- **Veritabanı:** MySQL Workbench
- **Mimari:** 3-Katmanlı Mimari (3-Tier Architecture)
