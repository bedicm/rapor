📋
Bu Hafta Neler Yapıldı?

Sipariş Takip Otomasyonu projesinin ilk haftasında, projenin temel yapı taşlarını oluşturmak ve yol haritasını belirlemek üzerine odaklanıldı.

✅
Yapılan Çalışmalar
Proje Tanımı
İşletmelerin sipariş takibini kolaylaştıracak bir masaüstü uygulamasının genel kapsamı çizildi.
Gereksinim Analizi
Sistemin temel fonksiyonları belirlendi: sipariş ekleme ve stok güncelleme.
Teknoloji Seçimi
Kullanıcı arayüzü için Java Swing, veritabanı yönetimi için MySQL Workbench kullanılmasına karar verildi.
Ortam Kurulumu
JDK, IntelliJ IDEA ve MySQL Server kurulumları tamamlanarak ilk bağlantı testleri yapıldı.
🎯
Önümüzdeki Haftanın Hedefleri

Gelecek hafta veritabanı şemasının (ER Diyagramı) tasarlanması ve Java ile MySQL arasındaki bağlantıyı sağlayacak JDBC katmanının oluşturulması hedeflenmektedir.
